 Please avoid excessive capitalization in bug reports. Also, suggestions belong on the suggestions forum: http://www.bay12forums.com/smf/index.php?board=5.0 [^]

For devs like Toady that don't develop on macOS, it is actually not that easy to make a DMG file. Also, a DMG is only a method of distributing an application, not running it.

I suspect the white screen issue is a duplicate of 0011680. Please try the solution mentioned there (PRINT_MODE:STANDARD) and reopen this issue if that doesn't fix it.

The wiki also has a page on other common issues you might run into: https://dwarffortresswiki.org/index.php/DF2014:Installation#Mac [^] - most of these have associated bug reports already.
