 That sounds like the "Truetype" font to me, which can be configured or turned off in data/init/init.txt:

Switches truetype on whenever the vertical tile size is this many pixels or larger.  You may also use 
YES/NO to force it on/off.  Turning it off may provide a minor speed boost (using font small enough that 
it doesn't activate won't).

[TRUETYPE:24]


You can set this to NO to disable it. Not a bug as-is.
